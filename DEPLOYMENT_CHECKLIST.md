# Django 博客系统部署检查清单

使用此清单确保你的部署顺利完成。

## 📋 部署前准备

### 代码准备
- [ ] 确保所有代码已提交到 Git 仓库
- [ ] 测试所有功能在本地正常运行
- [ ] 运行 `python manage.py check` 无错误
- [ ] 运行测试用例（如果有）

### 配置检查
- [ ] 修改 `SECRET_KEY` 为随机密钥
- [ ] 配置 `ALLOWED_HOSTS` 包含你的域名
- [ ] 设置 `DEBUG = False`（生产环境）
- [ ] 配置数据库连接信息
- [ ] 配置邮件服务（可选）

### 安全检查
- [ ] 生成强密码的超级用户
- [ ] 配置 HTTPS（生产环境必须）
- [ ] 设置文件上传大小限制
- [ ] 配置 CORS（如果需要）

## 🌐 CloudStudio 部署检查

### 基础配置
- [ ] 创建 CloudStudio 工作空间
- [ ] 选择 Python 3.9+ 环境
- [ ] 上传项目代码

### 环境设置
- [ ] 安装项目依赖：`pip install -r requirements.txt`
- [ ] 检查 Django 版本：`python -c "import django; print(django.VERSION)"`
- [ ] 检查数据库连接

### 数据库初始化
- [ ] 删除旧数据库（如果存在）
- [ ] 运行 makemigrations
- [ ] 运行 migrate
- [ ] 创建超级用户
- [ ] 记录管理员账号和密码

### 服务启动
- [ ] 配置端口映射（8000）
- [ ] 运行 `python manage.py runserver 0.0.0.0:8000`
- [ ] 访问公网地址测试
- [ ] 测试所有核心功能

## 🆘 故障排查

### 常见问题

#### 问题：无法访问网站
- [ ] 检查服务是否运行
- [ ] 检查防火墙设置
- [ ] 检查域名解析
- [ ] 检查 Nginx 配置

#### 问题：静态文件 404
- [ ] 运行 collectstatic
- [ ] 检查 STATIC_ROOT 配置
- [ ] 检查 Nginx 静态文件配置

#### 问题：数据库连接失败
- [ ] 检查数据库服务状态
- [ ] 检查连接配置
- [ ] 检查网络连接

#### 问题：上传文件失败
- [ ] 检查 MEDIA 目录权限
- [ ] 检查文件大小限制
- [ ] 检查磁盘空间

## 📞 获取帮助

如果遇到问题：

1. **查看日志**
   ```bash
   # Docker
   docker-compose logs

   # PythonAnywhere
   查看错误日志文件

   # CloudStudio
   查看终端输出
   ```

2. **检查配置**
   - 验证 settings.py 配置
   - 验证环境变量
   - 验证数据库连接

3. **查阅文档**
   - Django 官方文档
   - 本项目 DEPLOYMENT.md
   - 主机平台文档

4. **搜索解决方案**
   - Stack Overflow
   - Django 社区
   - 项目 Issues
